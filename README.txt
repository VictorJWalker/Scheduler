Title: FailsSource Scheduler
Purpose: The purpose of this application is to create a scheduling application for a fictional company.
         The application handles the adding, updating, and deleting of both Customer records and Appointment records.
         The application retrieves information from an SQL database, which handles timezone conversion and ID assignment.

Author: Victor Walker
Contact email: vwalk58@wgu.edu
IDE: IntelliJ IDEA Community Edition 2021.1.1 x64
JDK version: JDK 11.0.11
JavaFX Version: openJFX 11.0.2
SQL Driver: mysql-connector-java-8.0.25
How to run: Run the 'main' method, then input a valid username and password, stored in the SQL database.
Additional report: I chose to run a report counting the number of appointments scheduled for the day for all users.
                   The results of this report are displayed on the main menu.
